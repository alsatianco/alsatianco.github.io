Streamline

Infrastructure

Streamline is an event streaming platform that brings the power of Apache Kafka to small-to-medium workloads without the operational overhead. Single binary deployment, built-in web UI, and a familiar API for producers and consumers. Perfect for startups and indie projects that need reliable event streaming.
